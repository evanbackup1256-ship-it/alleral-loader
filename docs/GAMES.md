# Supported games

## Kick a Lucky Block

- **Script:** `games/kick_a_lucky_block.luau`
- **Data:** `games/data/kickblox.luau`
- **Features:** Auto kick, battle hosting, mastery, collect, weather, analytics webhooks

## Speed Keyboard Escape

- **Script:** `games/speed_keyboard_escape.luau`
- **Features:** Auto speed farming, keyboard automation

## Slime RNG

- **Script:** `games/slime_rng.luau`
- **Features:** Roll automation, inventory management

## Build A Ring Farm

- **Script:** `games/build_a_ring_farm.luau`
- **Features:** Planting, selling, upgrades, event farming

## Survive a Zombie Arena

- **Script:** `games/survive_a_zombie_arena.luau` (v2.0)
- **APIs ported from public scripts:**
  - `ZombieRemotes.ZombieDamage` — kill aura / instant kill
  - `WaveRemotes.SkipVote` — auto skip wave
  - Safe zone HipHeight + corner teleports
  - Auto shoot, god mode, ESP, aimbot, hitbox expander
  - Auto upgrade / buy / plant mine (remote probing)
  - Fly, noclip, walkspeed, FOV, auto farm preset
